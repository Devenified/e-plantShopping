# Paradise Nursery

Paradise Nursery is a React + Vite shopping cart application for browsing house plants and adding them to a cart.

## Features

- Landing page with company branding and a Get Started button
- Product listing page with plant categories and Add to Cart actions
- Shopping cart page with quantity updates, subtotals, totals, and delete actions
- Redux Toolkit state management for the shopping cart
- Responsive layout with remote image URLs for plant visuals

## Tech Stack

- React
- Vite
- React Router
- Redux Toolkit
- React Redux

## Run Locally

```bash
npm install
npm run dev
```
